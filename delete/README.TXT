JMETER Test zum löschen von Objekten (FOXML Dateien) in Fedora.
-----------------------------------------------------

Um den Test zu starten bitte run.sh ausführen. 

Der Test kann über die testplan.properties Datei 
konfiguriert werden. 

Die Datei testdaten.csv enthält eine Liste der PIDs der Objekte die geloescht werden sollen.
